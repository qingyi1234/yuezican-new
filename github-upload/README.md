# 月子餐个性化推荐网站

这是一个 Vite + React + TypeScript 静态网站，用于根据产后天数、分娩方式、哺乳状态、过敏忌口、常见病史和目标诉求，生成 42 天月子餐与康复训练建议。

## Vercel 部署设置

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`

## 本地预览

```bash
npm install
npm run build
npm run preview
```
